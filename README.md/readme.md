# SmartExpense 💰

SmartExpense is a full-stack personal expense management application that helps users track income, expenses, and categories while providing an easy-to-use dashboard for managing finances.

## Features

* 🔐 User Registration & Login (JWT Authentication)
* 👤 User Profile Management
* 💵 Income Management
* 💸 Expense Management
* 📂 Expense Categories
* 📊 Dashboard Overview
* ✏️ Edit & Delete Transactions
* 🌐 Responsive User Interface
* 🔒 Secure REST APIs

## Tech Stack

### Frontend

* React.js
* Vite
* Axios
* CSS

### Backend

* Java
* Spring Boot
* Spring Security
* JWT Authentication
* Maven

### Database

* MySQL

## Project Structure

```
SmartExpense/
│── backend/
│── frontend/
```

## Installation

### Backend

1. Open the `backend` folder.
2. Configure your MySQL database in `application.properties`.
3. Run:

```bash
mvn spring-boot:run
```

### Frontend

Open the `frontend` folder and run:

```bash
npm install
npm run dev
```
